# ✅ KPIs (Key Performance Indicators)
1. Total Points
Total points accumulated by all teams (or you can present top team’s points).
2. Average Goals Scored per Match
Measures attacking strength in the league.
3. Win Percentage (Top Team / Any Team)
Indicates the performance dominance of a team.
4. Goal Difference (Top Team / League Average)
Shows offensive + defensive efficiency combined.

# ✅ Business Analysis Questions
1. Which team is the top performer in the league based on total points?
Identifies the best overall performing club.
2. Which team has the strongest attack (highest goals scored)?
Shows the team with the best scoring ability.
3. Which team has the best defense (least goals conceded)?
Finds the most defensively strong club.
4. What is the relationship between wins and goal difference?
Helps analyze whether teams with higher goal difference also win more matches.
